Build-PorteoSite

Syntax 

Build-PorteoSite [-tenantUrl] <String> [-credentials] <Object> [-siteName] <String> [-title] <String> [-owner] <String> [[-type] <String>] [[-srcConnection] <Object>] [<CommonParameters>]

Parameters

Name Alias Description Required? Pipeline Input Default Value
-tenantUrl This parameter holds the tenant adminstrative site url. Ex. site-admin.sharepoint.com true false
-credentials This parameter holds the credentials for the tenant adminstrator. true false
-siteName This parameter holds the name of the site to be created true false
-title This parameter holds the title for the site to be created. true false
-owner This paramter holds the owner of the new site,email address; true false
-type This paramter holds the type of site to be created, TeamSite or CommunicationSite. false false TeamSite
-srcConnection This paramter holds the source to copy the template from. If not specified, no template is copied. false false

Examples

EXAMPLE 1 
$creds = Get-Credential
Gets the Credential for the subsequent examples
EXAMPLE 2 
$siteUrl = Build-PorteoSite -TenantUrl https://site-admin.sharepoint.com -Credentials $creds `
             -SiteName "SiteTemplate" -Title "The Site Template" -Type "TeamSite" `
             -Owner "name@site.onmicrosoft.com"
Builds the first site to use as a template. Now, modify this site adding to lists, pages, documents.
EXAMPLE 3 
$srcConn = Connect-PnPOnline -Url $siteUrl -Credentials $creds
Gets the connection for the template site from which you will copy.
EXAMPLE 4 
$dstCreds = Get-Credential
Gets the tenant credentials to to build the new site.
EXAMPLE 5 
$newSiteUrl = Build-PorteoSite -TennatUrl https://client-admin.sharepoint.com -Credentials $dstCreds `
             -SiteName "MySite" -Title "My Great Site" -Type "TeamSite" `
             -Owner "admin@company.com" -SrcConnection $srcConn
Builds the new site "MySite" from the template site on a different tenant.
