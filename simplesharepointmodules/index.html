SimpleSharepoint Modules

The following Powershell modules facilitate the provisioning of new sites on SimpleSharepoint's clients' tenants. The functionality is provided using two different modules, SSP.PorteoLibrary and SSP.PnPLibrary. The main functionality to be used is in the Build-PorteoSite function provided by the SSP.PorteoLibrary module. The Build-PorteoSite has two functions.
Build a base Porteo site
This function builds a base Porteo site with standard SimpleSharepoint lists. See the function for details on which lists are created. Then you would add content to this site to serve as provisioning new sites.
Copy and Provision a new Porteo Site from a source site.
This function will create a new site on a target tenant and copy the source site lists, pages, page contents, and site assets from the base site. Typically what you would do is create a base site using the above function first. Then, you would add content to that site, and once that is done and can be used to provision a new clients site, then you use this functionality to provision the newly created site.

SSP.PnPLibrary

This library contains somewhat primitive functions to provision sites and other things on a SharePoint tenant. The functions in this module are used by the SSP.PorteoLibrary to create common configured sites for a SimpleSharepoint tenant.

SSP.PorteoLibrary

This library contains functions to provision common SimpleSharepoint sites. The most notable function is be Build-PorteoSite
