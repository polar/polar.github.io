Add-SSPFiles

Uploades the files in a folder in the site/tennant.

This CmdLet uploads all the files that match the matches in a particular folder.

Syntax 

Add-SSPFiles [[-files] <Object>] [[-path] <String>] [[-folder] <String>] [[-connection] <Object>] [<CommonParameters>]

Parameters

Name Alias Description Required? Pipeline Input Default Value
-files This parameter contains the Files or File Names to upload from the specified path. false false
-path This parameter contains path from which to upload the files. false false
-folder This parameter contains the name/path of the folder in which to upload the files. false false
-connection This parameter provides the context for the call. The default is the current connection
returned by "Get-PnPConnection".		 false false (Get-PnPConnection)

Examples

EXAMPLE 1 
Add-SSPFile -Path "." -Files ("image1.png", "image2.jpg") -Folder "Site Assets"
EXAMPLE 2 
$files = Get-SSPFile -Path "." -Folder "Images" -Matches "*.jpg"
PS> Add-SSPFile -Path "." -Files $files -Folder "Site Assets"

Add-SSPSite

Adds a SOP Site Design using PnP to the site/tennant.

This CmdLet addes a new site design. If no Site Script Ids are defined then it adds
a bogus one, so that they may be set later.

Syntax 

Add-SSPSite [[-owner] <String>] [[-siteUrl] <String>] [[-siteName] <String>] [[-title] <String>] [[-type] <Object>] [[-siteDesignNames] <String[]>] [[-connection] <Object>] [<CommonParameters>]

Parameters

Name Alias Description Required? Pipeline Input Default Value
-owner false false
-siteUrl false false
-siteName This parameter contains the name of the Site. The name is supplied in its URL. false false
-title This parameter contains the title of the Site, which is displayed. false false
-type This parameter contains the type of site to create. It is either 'TeamSite' or 'CommunicationSite'.
The default is 'TeamSite'.		 false false TeamSite
-siteDesignNames false false
-connection This parameter provides the context for the call. The default is the current connection
returned by "Get-PnPConnection".		 false false (Get-PnPConnection)

destination url. It attempts to download image files from the source in
the Site Assets folder and upload them to the destination site.
